Buildroot is a simple, efficient and easy-to-use tool to generate embedded Linux systems through cross-compilation.

Buildroot is a set of Makefiles and patches that makes it easy to generate a complete embedded Linux system.

Can handle everything

Cross-compiling toolchain, root filesystem generation, kernel image compilation and bootloader compilation.

Is very easy

Thanks to its menuconfig, gconfig and xconfig configuration interfaces, building a basic system with Buildroot typically takes 15-30 minutes.

Supports several hundreds of packages

X.org stack, Gtk2, Qt, DirectFB, SDL, GStreamer, a large number of network-related and system-related utilities are supported.

Buildroot is for Everyone.

Has a simple structure that makes it easy to understand and extend. It relies only on the well-known Makefile language.

Our recent activity.

Buildroot is an open source project: many developers contribute to it daily.
